# display 属性

# 概述

在 html 文档中，每个 html 元素都会对应一个盒子，那么这个盒子是由谁生成的呢？答案是元素的 display 属性。

# 一个盒子，两个特性

display 属性不仅为每个 html 元素生成一个盒子，还会为盒子添加外部特性和内部特性。

# 一个盒子

display 属性为 html 元素生成的盒子，最直观的展示就是我们平时所说的盒子模型。如下图所示：

盒子模型相关的一些术语：

  • content edge: 图中已经明确标出
  • content box: content edge 框出来的矩形区域
  • padding edge: 图中已经明确标出
  • padding box: padding edge 框出来的矩形区域
  • padding area: padding box 挖去 content box 剩下的环形框
  • border edge: 图中已经明确标出
  • border box: border edge 框出来的矩形区域
  • border area: border box 挖去 padding box 剩下的环形框
  • margin edge: 图中已经明确标出
  • margin box: margin edge 框出来的矩形区域
  • margin area: margin box 挖去 border box 剩下的环形框

TIP

box + block-level + block container = block box

box + inline-level + IFC = inline box

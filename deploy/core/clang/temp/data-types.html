# 字符型范围

有的编译将 char 类型实现为无符号的，此时它能表示的范围就是 0 ~ 255 。也有的编译器将 char 类型实现为有符号的，则它能表示的范围就是 -128 ~ 127 。

# 类型长度

类型的长度可以通过 stdio.h 头文件中的 sizeof 函数获取到。如：

#include <stdio.h>

int main(int argc, char const *argv[])
{
  // 打印当前系统 int 类型的长度
  printf("length of int is %d bytes\n", sizeof(int));

  return 0;
}

// output
length of int is 4 bytes

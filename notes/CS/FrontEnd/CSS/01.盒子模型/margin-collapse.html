# margin 重叠

# 满足什么条件的两个元素会发生 margin 重叠

在同一个 BFC 中的两个 block-level box ，它们的纵向 margin 之间没有被 line box ，padding ，border ，clearance 隔开，那么它们的纵向 margin 会发生重叠

# 建造者模式

# 概述

我们已经介绍了四种创建对象的模式，分别为：

  • 单例模式
  • 简单工厂模式
  • 工厂方法模式
  • 抽象工厂模式

这四种模式在创建对象时有一个共同点，就是 new 一个完整的对象。但是有的时候我们 new 一个对象时，希望对这个对象的组成有自己的控制。比如同是建造一辆汽车，大众汽车厂在建造这辆汽车的时候，想要使用德国产的发动机，丰田汽车厂在建造这辆汽车时想使用日本产的发动机。如果使用上述四种模式，想对建造汽车过程中所使用的零部件进行细分，很难做到。这就引出了建造者模式。

# 示例

现在先建造一辆汽车，我们分以下几步来进行：

  • 建造骨架
  • 建造发动机
  • 建造车轮
  • 装配成一辆整车

我们把这个过程抽象为 CarBuilder

public abstract class CarBuilder {

    public abstract void buildFrame();

    public abstract void buildEngine();

    public abstract void buildWheel();

    public abstract Car getCar();
}

最终将建造出来的汽车定义为 Car

public class Car {
    private Frame frame;
    private Engine engine;
    private Wheel wheel;

    // getter setter
}

我们把一辆车分为了三个大的部件，分别为：骨架，发动机和车轮。每一个部件都有很多种，如下：

  • 骨架
    • 中国产的骨架
    • 日本产的骨架
    • 美国产的骨架
  • 发动机
    • 德国产的发动机
    • 日本产的发动机
    • 英国产的发动机
  • 车轮
    • 中国产的车轮
    • 日本产的车轮
    • 法国产的车轮

部件的定义略。

大众汽车厂在建造这辆汽车时，它选择的是：

  • 中国产的骨架（性价比高）
  • 德国产的发动机（性能好）
  • 中国产的车轮（性价比好）

定义为 DashAutoCarBuilder

public class DashAutoCarBuilder extends CarBuilder {

    private Car car = new Car();

    @Override
    public void buildFrame() {
        car.setFrame(new ChineseFrame());
    }

    @Override
    public void buildEngine() {
        car.setEngine(new GermanyEngine());
    }

    @Override
    public void buildWheel() {
        car.setWheel(new ChineseWheel());
    }

    @Override
    public Car getCar() {
        return car;
    }
}

丰田汽车厂在建造这辆汽车时，它选择的是：

  • 日本产的骨架（支持本国产品）
  • 日本产的发动机（支持本国产品）
  • 日本产的车轮（支持本国产品）

定义为 ToyotaCarBuilder

public class ToyotaCarBuilder extends CarBuilder {

    private Car car = new Car();

    @Override
    public void buildFrame() {
        car.setFrame(new JapaneseFrame());
    }

    @Override
    public void buildEngine() {
        car.setEngine(new JapaneseEngine());
    }

    @Override
    public void buildWheel() {
        car.setWheel(new JapaneseWheel());
    }

    @Override
    public Car getCar() {
        return car;
    }
}

我们还需要一个 Director ，接收具体的 builder ，根据不同的 builder 来建造有着不同部件的汽车。

public class Director {
    private CarBuilder carBuilder;

    public Director() {
    }

    public Director(CarBuilder carBuilder) {
        this.carBuilder = carBuilder;
    }

    public Car build() {
        carBuilder.buildFrame();
        carBuilder.buildEngine();
        carBuilder.buildWheel();
        return carBuilder.getCar();
    }
}

测试

public class Test {
    public static void main(String[] args) {
        // 大众
        Car dashAutoCar = new Director(new DashAutoCarBuilder()).build();
        System.out.println(dashAutoCar);

        // 丰田
        Car toyotaCar = new Director(new ToyotaCarBuilder()).build();
        System.out.println(toyotaCar);
    }
}

# Redis 示例

package com.yungsem.springbootstart.builder;

/**
 * @author ys
 * @version 1.0
 */
public class Redis {
    private String host;
    private int port;
    private int database;
    private String password;

    public Redis() {
    }

    public String getHost() {
        return host;
    }

    public void setHost(String host) {
        this.host = host;
    }

    public int getPort() {
        return port;
    }

    public void setPort(int port) {
        this.port = port;
    }

    public int getDatabase() {
        return database;
    }

    public void setDatabase(int database) {
        this.database = database;
    }

    public String getPassword() {
        return password;
    }

    public void setPassword(String password) {
        this.password = password;
    }

    @Override
    public String toString() {
        return "Redis{" +
                "host='" + host + '\'' +
                ", port=" + port +
                ", database=" + database +
                ", password='" + password + '\'' +
                '}';
    }

    public static Builder builder() {
        return new Builder();
    }

    public static class Builder {
        private String host;
        private int port;
        private int database;
        private String password;

        public Builder() {
        }

        public Builder withHost(String host) {
            this.host = host;
            return this;
        }

        public Builder withPort(int port) {
            this.port = port;
            return this;
        }

        public Builder withDatabase(int database) {
            this.database = database;
            return this;
        }

        public Builder withPassword(String password) {
            this.password = password;
            return this;
        }

        public Redis build() {
            Redis redis = new Redis();
            redis.setHost(host);
            redis.setDatabase(database);
            redis.setPort(port);
            redis.setPassword(password);
            return redis;
        }
    }
}

public static void main(String[] args) {
    Redis redis = Redis.builder()
            .withHost("localhost")
            .withPort(6379)
            .withDatabase(2)
            .withPassword("123")
            .build();

    System.out.println(redis.toString());
}

output:

Redis{host='localhost', port=6379, database=2, password='123'}

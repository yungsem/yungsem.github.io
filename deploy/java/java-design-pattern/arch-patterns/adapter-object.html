# 适配器模式：对象适配器

对象适配器就是类适配器的改版，如下：

Target 接口没变：

public interface Target {
    void request();
}

Adaptee 类也没变：

public class Adaptee {
    public void specificRequest() {
        System.out.println("=====>>>specificRequest from Adaptee");
    }
}

Adapter 类依旧实现 Target 接口，但不再继承 Adaptee 类了，而是直接注入 Adaptee 对象，如下：

public class Adapter implements Target {
    /**
     * Adaptee 对象
     */
    private Adaptee adaptee;

    public Adapter() {
    }

    /**
     * 通过构造函数注入 Adaptee 对象
     * @param adaptee Adaptee 对象
     */
    public Adapter(Adaptee adaptee) {
        this.adaptee = adaptee;
    }

    @Override
    public void request() {
        adaptee.specificRequest();
    }
}

使用：

// 使用时需要注入 Adaptee 对象
Target target = new Adapter(new Adaptee());
target.request(); // =====>>>specificRequest from Adaptee

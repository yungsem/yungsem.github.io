# Containing block

# 概述

containing block 是 CSS 中非常重要的一个概念。BFC 和 IFC 对盒子的约束规则就是依赖于 containing block 而定的。元素的 position 定位也以 containing block 为基础。

# 元素生成的 containing block 和元素的 containing block

下面两个概念：

  • 一个元素生成的 containing block (the containing block established by the element)
  • 该元素的 containing block (the containing block of the element)

是两个不同的概念，不要混淆。

一个元素生成的 containing block 是指该元素为其子孙元素生成的 containing block 。而该元素的 containing block 是指该元素的祖先元素为该元素自身生成的 containing block 。

下面我们要重点讲述的是怎样确定一个元素的 containing block （注意不是一个元素生成的 containing block）。

为了表述精准，我们在描述一个元素的 containing block 时，通常表述为：“A 元素的 containing block 是 B 元素的 XXX box 对应的矩形框”，我们也说：“A 元素的 containing block 是由 B 元素生成的”。

# Containing Block

A 元素的 containing block 按照以下规则确定：

  • 如果 A 元素是根元素（在 html 文档中，根元素就是 html 元素），则它的 containing block 是浏览器的视口（view port）。
  • 如果 A 元素不是根元素
    • 如果 A 元素的 position 属性值是 fixed ，则它的 containing block 是浏览器的视口（view port）。
    • 如果 A 元素的 position 属性值是 relative 或 static ，则它的 containing block 是离 A 元素最近的，有 block container box 的祖先元素的 content box 对应的矩形框。
    • 如果 A 元素的 position 属性值是 absolute ，则它的 containing block 由离 A 元素最近的，position 属性值由 absolute, relative 或 fixed 的祖先元素（下面称为 B 元素）生成，但是具体是由 B 元素的哪个 box 确定，需要遵循以下规则：
      • 如果 B 元素是一个 block-level element ，那么 A 元素的 containing block 就是 B 元素的 padding box 对应的矩形框。
      • 如果 B 元素是一个 inline-level element ，
        • 如果 B 元素占据一行，那么 A 元素的 containing block 就是 B 元素的 padding box 对应的矩形框。（见示例一）
        • 如果 B 元素占据多行，那么 A 元素的 containing block 就是 B 元素的第一行的 left padding edge ，top padding edge 和 B 元素的最后一行的 right padding edge ，bottom padding edge 组成的矩形框。（见示例二）

# 说明示例

# 示例一（ele-b 占据一行）

初始状态：

.ele-b {
    position: absolute;
    border: 2px dotted black;
    padding: 20px;
}
.ele-a {
    border: 1px solid red;
}

<span class="ele-b">
    <span>文字</span><span class="ele-a">x</span><span>文字</span>
</span>

设置 ele-a 的 position 为 absolute：

  • 黑色虚线的矩形框就是 ele-b 的 padding box
  • ele-a 的 containing block 就是黑色虚线的矩形框
  • ele-a 脱离文档流
.ele-b {
    position: absolute;
    border: 2px dotted black;
    padding: 20px;
}
.ele-a {
    border: 1px solid red;
    position: absolute;
}

<span class="ele-b">
    <span>文字</span><span class="ele-a">x</span><span>文字</span>
</span>

设置了 top 和 left：

  • ele-a 以 containting block 为基准，重新定位
.ele-b {
    position: absolute;
    border: 2px dotted black;
    padding: 20px;
}
.ele-a {
    border: 1px solid red;
    position: absolute;
    top: 5px;
    left: 5px;
}

<span class="ele-b">
    <span>文字</span><span class="ele-a">x</span><span>文字</span>
</span>

# 示例二（ele-b 占据多行）

初始状态：

.wrapper {
    width: 400px;
    padding: 100px;
}
.ele-b {
    position: relative;
    border: 2px dotted black;
    padding: 20px;
}
.ele-a {
    border: 1px solid red;
}

<div class="wrapper">
    <span class="ele-b">
        <span>文字文字文字文字文字文字文字文字文字文字文字文字文字文字</span><span class="ele-a">x</span><span>文字文字文字文字文字文字文字文字文字文字文字文字文字文字</span>
    </span>
</div>

设置 ele-a 的 position 为 absolute：

.wrapper {
    width: 400px;
    padding: 100px;
}
.ele-b {
    position: relative;
    border: 2px dotted black;
    padding: 20px;
}
.ele-a {
    border: 1px solid red;
    position: absolute;
}

<div class="wrapper">
    <span class="ele-b">
        <span>文字文字文字文字文字文字文字文字文字文字文字文字文字文字</span><span class="ele-a">x</span><span>文字文字文字文字文字文字文字文字文字文字文字文字文字文字</span>
    </span>
</div>

设置了 top, bottom, left, right：

.wrapper {
    width: 400px;
    padding: 100px;
}
.ele-b {
    position: relative;
    border: 2px dotted black;
    padding: 20px;
}
.ele-a {
    border: 1px solid red;
    position: absolute;
    top: 5px;
    bottom: 5px;
    left: 5px;
    right: 5px;
}

<div class="wrapper">
    <span class="ele-b">
        <span>文字文字文字文字文字文字文字文字文字文字文字文字文字文字</span><span class="ele-a">x</span><span>文字文字文字文字文字文字文字文字文字文字文字文字文字文字</span>
    </span>
</div>

# 权重

# 几个概念

p {
    color: red;
}

其中：

  • p 称为选择器
  • color: red 称为样式声明（简称样式）
  • color 称为样式属性（简称属性）
  • red 称为样式属性值（简称值）

# 概述

CSS 权重主要用于解决样式冲突问题。每一条样式声明都有自己的权重。当多条属性相同的样式声明应用到同一个元素上时，权重最大的那条样式声明胜出。

一条样式声明的权重由以下四个方面组成：

  • weight
  • origin
  • specificity
  • order

权重比较的规则（The cascade rules）如下：

  1. 比较 weight
    • important declarations > nonimportant declarations
  2. 比较 origin
    • user agent declarations
    • user normal declarations
    • author normal declarations
    • author important declarations
    • user important declarations
  3. 比较 specificity
    • specificity 值大的 > specificity 值小的
  4. 比较 order
    • 后声明的 > 先声明的

# Weight

可以在某个样式声明结尾的分号前加上 !important 标识。这样，这个样式声明就称为 important declaration。

p.dark {
    color: #FFF !important; /*important declaration*/
    background: white;
}

important declaration 没有sepcificity 值。

# Origin

略

# Specificity

sprecificity 是针对 selector 而言的。每一个 selector 都有一个 specificity 值。某一条样式声明的 specificity 值就是这条样式声明对应的 selector 的 specificity 值。

计算一个 selector 的 specificity 值所依据的规则如下：

  • id 值 0, 1, 0, 0
  • class 值，属性值，伪类值 0, 0, 1, 0
  • 元素，伪元素 0, 0, 0, 1
  • 通配符 0, 0, 0, 0
  • 连接符 have no specificity

注意：内联样式的 specificity 值是1, 0, 0, 0

sample:

/* specificity = 0, 1, 0, 0*/
#id216 {
    color: blue
} 

/* specificity = 0, 0, 1, 0*/
.grape {
    color: purple
} 

/* specificity = 0, 0, 1, 1*/
tr[id="totals"] {
    color: purple
} 

/* specificity = 0, 0, 1, 1*/
a:link {
    color: purple
} 

/* specificity = 0, 0, 0, 2*/
p em {
    color: puple
}

# Order

一个 html文档所包含的 css 样式有以下几种，它们的顺序是：

  1. 外部导入进来的样式
  2. 在当前文档中用<style></style>标签声明的样式
  3. 内联样式

示例：

// demo.css
p {
    color: red;
}

// demo.html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>demo</title>
    <link rel="stylesheet" href="demo.css">
    <style>
        p {
            color: blue;
        }
    </style>
</head>
<body>
    <p style="color: black;">This is just a test.</p>
</body>
</html>

# Inheritance

什么是样式继承

举例说明：如果对某个<h1>元素声明了如下样式：

h1 {
    color: gray;
}

那么这个<h1>元素所有的后代元素都具有了样式:

{
    color: gray;
}

这样的继承机制也被称为样式向下传递。注意，样式只能向下传递，不能向上传递。

但是有个例外，<body>元素的样式可以向上传递给<html>元素。

有些样式不会向下传递

以下样式不会向下传递：

  • border
  • margin
  • padding
  • background

继承而来的样式没有specificity值

继承而来的样式没有specificity值。

在下面的例子中，通配符使<em>元素具有了样式1：

{
    color: gray; /*0, 0, 0, 0*/
}

继承机制使<em>元素具有了样式2：

{
    color: black; /*没有specificity值*/
}

由于0specificity值大于没有sepcificity值，样式1胜出，所以Central显示为灰色。

* {
    color: gray;
}
h1#page-title {
    color: black;
}

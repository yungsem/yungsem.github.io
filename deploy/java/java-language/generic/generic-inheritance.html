# 泛型类型的继承关系

# Type Parameter 的继承关系不代表 Generic Type 的继承关系

Double 和 Integer 是 Number 的子类型。那么 Box<Double> 和 Box<Integer> 是不是 Box<Number> 的子类型呢？

答案是：否。

事实上，Box<Double> 和 Box<Integer> 的父类型是 Object 。

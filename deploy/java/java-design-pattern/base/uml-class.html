# 单向 1 对 1

一个顾客（Customer）只有一个账户（Account）：

public class Customer {
  private Account account;
}

# 单向 1 对 多

一个顾客（Customer）拥有多个地址（Address）：

public class Customer {
  private List<Address> addresses;
}

# 聚合（Aggregation）

一辆汽车（Car）中包含一个引擎（Engine）。如果汽车不在了，引擎还可以继续存在：

public class Car {
  private Engine engine;

  public void setEngine(Engine engine) {
    this.engine = engine;
  }
}

# 组合（Composition）

一个公司（Company）可以有一个或多个部门（Department）。如果公司不在了，部门也会消失：

public class Car {
  private Department dept = new Department();
}
